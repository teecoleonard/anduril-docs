# Reajustes de Contratos

Documentação completa sobre o sistema de reajustes de valores em contratos.

## Documentos Disponíveis

- **[Arquitetura](./arquitetura)** - Diagramas e fluxos
- **[Planejamento Longo Prazo](./longo-prazo)** - Estratégia de 30 anos

## Tópicos Principais

- Cálculo de saldo devedor
- Progressão de reajustes
- Integração com boletos
- Sincronização de dados

## Como Aplicar Reajuste

Entre na página de contratos, veja qual contrato será aplicado o reajuste e clique no botão escrito **Reajuste**, segue imagem exemplo:
