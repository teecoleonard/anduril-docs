# Como Cadastrar Empresa

## Objetivo
Fazer o vinculo da empresa em conjunto com o sistema ERP


## Pré-requisitos
- Dados da empresa disponíveis
- Dados do banco disponíveis

## Localização
**Dashboard → Empresa**

## Passo-a-Passo

### **Etapa 1: Acessar a Tela de Novo Contrato**

1. Acesse **"Empresa"** no menu
2. Preencha todos os dados de identificação da empresa

> ![alt text](image.png)


3. Preencha os dados de configuração SICOOB

> ![alt text](image-1.png)

4. Clique em "Salvar Dados"

## **Considerações**

Atualmente o sistema apenas suporta os dados de configuração do **SICOOB** eles seram necessário para que o **gerador de CNAB** funcione.
